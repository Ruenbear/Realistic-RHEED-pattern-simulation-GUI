## 坐标系规定

- x axis: 沿着电子枪入射方向
- y axis：水平方向
- z axis：竖直方向



## 参数输入

### EXP

- Distance: 样品与荧光屏的距离

- Voltage: 电子枪的加速电压
- Tilt：面外角
- Phi：面内角



### OPT

- Screen Size: 荧光屏的尺寸
- Pixel size: 模拟的精度，指一个像素点对应的实际荧光屏的长度
- Color： 颜色映射序号



### SYS

- Lattice const: 晶体沿着三个非共线方向的晶格常数。
- Direction Vec: 方向矢量，不需要归一
- Num：该方向上的原胞数量
